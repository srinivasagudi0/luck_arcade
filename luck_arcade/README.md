# Luck Arcade

Luck Arcade is a Streamlit-based application that offers a collection of fun and interactive games of chance. This project is designed for entertainment and learning purposes, providing a simple interface for users to play and enjoy various luck-based games.

## Features
- Multiple games of chance implemented in Python
- User-friendly Streamlit web interface
- Easy to extend with new games

## Getting Started

### Prerequisites
- Python 3.7+
- [Streamlit](https://streamlit.io/)

### Installation
1. Clone the repository or copy the `luck_arcade` folder into your project.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
   (If `requirements.txt` is not present, install Streamlit manually: `pip install streamlit`)

### Running the App
Navigate to the `luck_arcade` directory and run:
```bash
streamlit run streamlit_app.py
```

## Folder Structure
- `streamlit_app.py` - Main Streamlit application file
- `cli.py` - Command-line interface for the arcade (if available)
- `__init__.py` - Package initialization
